POGIL: List of objectives

The topic for this discussion is 'Brainstorm a list of topics for our extra credit discusions'

  1. Basic Data types
    1. What can each type store?
    2. Good choices for different information / user input?
    3. Create a 'rules of thumb' / decision tree for which type to use
  2. Expression Evaluation
    1. Students will determine (and explain) how a given C# expression is evaluated
      1. Students will show intermediate results, in the order of evaluation
      2. Students will evaluate an expression without showing the intermediate results
    2. Students will construct new expressions that represents
      1. mathematical formulas
      2. verbal / written descriptions
